📚 Documentación de la API

Esta aplicación proporciona una API REST completa. Puedes acceder a:

GET /docs - Documentación interactiva (Swagger UI)
GET /redoc - Documentación alternativa (ReDoc)
GET /users/ - Listar todos los usuarios
GET /pets/ - Listar todas las mascotas
GET /flights/ - Listar todos los vuelos
POST /flights/search/ - Buscar vuelos disponibles
GET /reservations/ - Listar todas las reservas
